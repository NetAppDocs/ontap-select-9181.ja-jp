= Deploy 管理者パスワードの変更

コマンドラインインターフェイスを使用して、必要に応じてDeploy仮想マシン管理者アカウントのパスワードを変更できます。

.手順
. 管理者アカウントを使用して、DeployユーティリティのCLIにSign inします。
. パスワードを変更してください：
`password modify`
. 環境に応じて、すべてのプロンプトに適切に対応してください。
