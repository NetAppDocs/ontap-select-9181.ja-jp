= ONTAP Select ビデオコンソールにアクセスします
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP Selectが実行されているハイパーバイザー仮想マシンのビデオコンソールにアクセスできます。

.タスク概要
問題のトラブルシューティングを行う場合や、NetAppサポートからの要請があった場合に、仮想マシンのコンソールにアクセスしなければならないことがあります。

.手順
. vSphereクライアントにアクセスしてSign inします。
. 階層内の適切な場所に移動して、ONTAP Select仮想マシンを特定します。
. 仮想マシンを右クリックして、*コンソールを開く*を選択します。
