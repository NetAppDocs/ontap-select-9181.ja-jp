= ONTAP Selectライセンスの購入時のワークフロー
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
以下のワークフローは、ONTAP Select環境のライセンスを購入して適用するプロセスを示しています。ライセンスを購入する際は、ライセンス モデルとストレージ容量を選択する必要があります。

具体的な手順は、Capacity Tierライセンスを使用しているか、Capacity Poolライセンスを使用しているかによって異なります。

9桁のライセンスシリアル番号:: シリアル番号は、ノード（Capacity Tiers）またはストレージプール（Capacity Pools）のいずれかに適用されます。
ライセンス ロック ID:: Capacity Poolライセンスを使用する場合は、Deployインスタンスのライセンスロック IDが必要です。
ライセンス Web サイト:: 容量ティアと容量プールのライセンスは、それぞれ異なるWebサイトで取得します
